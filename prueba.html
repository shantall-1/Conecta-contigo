Test de Salud Mental

Responde honestamente. Este test no reemplaza un diagnóstico profesional.

1. ¿Te has sentido triste o desmotivado últimamente?

2. ¿Tienes problemas para dormir?

3. ¿Sientes ansiedad o nerviosismo durante el día?

4. ¿Te cuesta concentrarte o mantener el enfoque?

5. ¿Te sientes cansado incluso después de descansar?
